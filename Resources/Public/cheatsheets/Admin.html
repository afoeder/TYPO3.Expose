Admin sheet

Class Annotations
Active
makes this entity visible to admin
Group("name")
specify a group for this entity.By default the Group is the name of the entities Package
Label ("label")
specify a label to use in the navigation for the entity
Set (title="", properties="")
use this to group the properties into fieldsets
Variant (variant="Calendar", options="Calendar,List")
specify a different variant for this entity and if desired variants to switch between as options
Varian­tMa­pping (key="property",...)
this helps the variants to access the right properties
Controller Action Annotations
Access (role="editor", admin="true")
restricts the access to this controller
Navigation (title="", position="top|left", priority="10")
adds the action to the specific navigation
Example
This adds the someAction to the fixed Navigation at the top and restricts the access to the users with the editor role 
										/**
										 *@Admin\Access(role="editor")
										 *@Admin\Navigation(title="Foo",position="top")
										 **/
										public function someAction(){...}
Property Annotations
Editor ("rte|markdown|textile|...")
set the editor for the Textarea Widget
Filter
make this property a filter in the list view
Ignore (["list,inline"])
ignore this property alltog­ether or in specific views
InfoText ("This field....")
additional info text below the input
Inline
make the related entities inline editable
Label ("label")
set a specific label for the property
Option­sPr­ovider ("Array|Relation|Policy")
use a different Option­sPr­ovider
Representation (datetimeFormat="")
change the format for the DateJs widget
Title
define one or more properties as title for the entity
Variant ("Stacked|Inline")
change the variant for the inline editor
Widget ("Textarea")
specify a different widget for the property
Examples
Entity 
										/**
										  *@Admin\Active
										  *@Admin\Group("Blog")
										  *@Admin\Label("Posts")
										  *@Admin\Set(title="Post", properties="title,content")
										  *@Admin\Set(title="Meta", properties="author,tags,...")
										  *@Admin\Variant(variant="List", options="List,Calendar")
										  *@Admin\VariantMapping(title="title", start="date")
										 **/
										class Post {...}
Simple Property 
										/**
										  *@Admin\Label("Post Author")
										  *@Admin\Filter
										  *@Admin\InfoText("The name of the author")
										  *@Admin\Title
										 **/
										protected $author;
Relations 
										/**
										 * @var \Doctrine\…\Collection<\TYPO3\Blog\…\Post>
										 * @ORM\ManyToMany
										 * @Admin\Widget("Chosen")
										 */
										protected $relatedPosts;
									
										/**
										 * @var \Doctrine\…\Collection<\TYPO3\Blog\…\Comment>
										 * @ORM\OneToMany(mappedBy="post")
										 * @Admin\Ignore("list")
										 * @Admin\Inline
										 * @Admin\Variant("Tabular")
										 */
										protected $comments;
